# AI-Powered Vector Database Query Profiler

## Technologies Used
- FastAPI
- SQLite
- ChromaDB
- Sentence Transformers
- HTML/CSS/JavaScript

## Features
- SQL Query Profiling
- Vector Search Profiling
- Hardware Monitoring
- Query Logging
- File Upload Support

## Workflow
1. User enters SQL and Vector queries.
2. FastAPI receives the request.
3. SQL query runs on SQLite.
4. Text query is converted into embeddings.
5. ChromaDB performs vector search.
6. Hardware metrics are collected.
7. Results and performance metrics are displayed.
8. Query logs are saved for analysis.
